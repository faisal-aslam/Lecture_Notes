% PDFs into one combined "full course pack" PDF using pdfpages.

% fully independent (own title page, own numbering) and this
% file just merges the finished PDFs together, in order.
%
% Usage:
%   1. Compile each lecture file individually first
%      (e.g. pdflatex lecture1-math-foundations.tex), so each
%      lectureN-*.pdf exists in this folder.
%   2. Add each new lecture's PDF filename to the \includepdf
%      list below, in order.
%   3. Compile this file to produce full-course-notes.pdf.
% ============================================================
